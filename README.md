# Vehicle Test Data Analysis

## Overview
This project analyzes simulated vehicle longitudinal test data using Python. The objective was to build a simple validation-style workflow for handling raw signals, cleaning noisy data, calculating engineering KPIs, and generating plots for test-run comparison.

## Objective
To simulate an automotive vehicle test data analysis workflow relevant to vehicle testing, validation, and e-mobility engineering roles.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Signals Included
- Time
- Speed
- Acceleration
- Throttle
- Brake
- Battery voltage
- Battery current
- Motor torque
- Distance
- Ambient temperature
- Battery power

## Workflow
1. Generated simulated vehicle test data
2. Cleaned missing and noisy values
3. Applied signal smoothing
4. Calculated key performance indicators
5. Generated engineering plots
6. Exported KPI summary as CSV

## KPIs Calculated
- Maximum speed
- Average speed
- Peak acceleration
- Peak deceleration
- 0–50 km/h time
- Braking distance
- Estimated energy usage
- Maximum motor torque

## Outputs
- Cleaned CSV files
- KPI summary table
- Speed vs Time plots
- Acceleration vs Time plots
- Battery Power vs Time plots

## Note
The dataset used in this project is simulated for portfolio and learning purposes and does not represent proprietary OEM test data.
